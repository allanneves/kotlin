# Kotlin CheatSheet


This repo has been created to serve as a cheatsheet for the Kotlin Language and it is composed by reference implementations, exercises and practices collected by me throughout the completion of the course [Complete Kotlin Masterclass](https://www.udemy.com/course/kotlinmasterclass/).

It covers the main points from the Basics to more advanced topics like Coroutines. Android development and functional programming with Arrow are not part of this repository. Expect to find mainly code covering each topic, any type of explanation has been added in form of code comment and does not aim to do a better job than the official docs at Kotlin's Official [Website](https://kotlinlang.org/)
